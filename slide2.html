STEP 01

지갑 생성하기

Create New Wallet

1

웹사이트 접속 및 앱 실행

1-1. Chrome 앱스토어에서 Daipha Wallet을 검색하여 Chrome에 추가를 클릭하여 설치합니다.

1-2. 크롬 상단에 확장프로그램 아이콘 이 생성되면 클릭하여 Create a wallet을 선택합니다.

2

새 지갑 만들기 선택

초기 화면에서 [Create New Wallet] 버튼을 탭하여 새로운 지갑 생성 프로세스를 시작합니다.

3

비밀번호 설정

비밀번호 생성 규칙(최소 8자, 대소문자 포함 등)을 확인하고 안전한 비밀번호를 입력한 후 Next 버튼을 클릭합니다.

4

니모닉 구문 백업 (Mnemonic Phrase)

15개의 단어 복구 구문을 안전한 장소에 기록합니다.

경고: 분실 시 영원히 자산을 잃게 됩니다.

5

니모닉 구문 확인 (Confirm Phrase)

시스템이 요청하는 특정 단어를 순서대로 입력하면 지갑이 생성됩니다.
